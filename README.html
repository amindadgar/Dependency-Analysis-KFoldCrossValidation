Reproduction instruction#

To reproduce the results

  • First run pip install -r requirments.txt.

  • Download each datasets below, move and extract them in a folder named Datasets.

    1. Liver disorders Path should be Datasets/liver-disorders/bupa.data

    2. Letter recognition: Path should be Datasets/letter recognition/letter-recognition.data

    3. MAGIC gamma telescope data: Path should be Datasets/magic/magic04.data

    4. Page-Blocks dataset: Path should be Datasets/page-blocks/page-blocks.data

    5. Wine Quality dataset: Pathes should be

      • Datasets/wine-quality/winequality-red.csv

      • Datasets/wine-quality/winequality-white.csv

    6. Habermanâ€™s survival dataset: Path should be Datasets/haberman/haberman.data

  • After that remove the folder results or move it to another location and make an empty folder with a same name.

  • Finally run the all the cells in the main.ipynb notebook and wait for it to be finished.
